## 📌 ‘단건 조회’와 ‘목록 조회’

### 🔹 단건 조회 (Single Resource)

```http
GET /posts/15
````

* 의미: **15번 게시글 하나만 조회**

👉 ‘단건 조회’는 데이터가 여러 개 있는 상황에서
**특정 하나의 리소스를 조회하는 API**

👉 즉, 여러 개 중에서 **하나만 가져오는 것**

---

### 🔹 목록 조회 (Collection)

```http
GET /posts
```

* 의미: **게시글 여러 개 조회**

👉 이는 단건 조회가 아니라
**여러 개의 데이터를 가져오는 ‘목록 조회’**


#### 정리
👉 단건 조회(데이터 식별자)는 **Path Variable**을 사용,  
👉 목록 조회는 필요에 따라 **Query String으로 조건을 추가**
````
